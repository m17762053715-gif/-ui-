真正高质量的实现

我们深入研究了 React Bits 的核心算法， 理解了每个组件背后的数学原理和着色器实现， 然后用纯 WebGL 完整重写。每个组件都基于真实源码， 保留了原版的视觉质量和算法精髓。

Galaxy - 星系 WebGL + Hash 函数

使用 Hash 函数生成伪随机星星分布，多层深度系统创造 3D 视差效果。 星星会闪烁、旋转，并对鼠标产生排斥反应。完全基于 React Bits 原版算法。

Aurora - 极光 WebGL 2.0 + Simplex Noise

核心：Simplex Noise 噪声函数。 通过指数函数和 smoothstep 创造柔和的极光过渡效果。颜色渐变系统支持 3 个颜色停止点。

Waves - 波浪 Canvas 2D + Perlin Noise

核心：Perlin Noise + 物理弹簧系统。 每个点通过 Perlin Noise 驱动，加上弹簧物理模拟和鼠标交互，创造流动的波浪效果。

Plasma - 等离子体 WebGL 2.0 + 复杂着色器

核心：复杂的数学着色器。 使用 ray marching、正弦波叠加、tanh 函数创造流动的等离子体效果。支持彩色和单色模式。

Particles3D - 3D 粒子 WebGL + 3D 相机

核心：3D 透视投影 + 正弦波动画。 粒子在 3D 空间中通过正弦波运动，使用透视投影实现近大远小效果。完整的 3D 相机系统。
